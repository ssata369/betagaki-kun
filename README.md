# ベタガキ君

ノーコードで LP・EC コンテンツを作成し、**CMS にそのまま貼り付けられる HTML コード**を出力するブラウザツールです。
エンジニアでなくても、パーツをクリック／ドラッグして組み立て、出力したコードを CMS の所定欄に貼るだけでページを再現できます。

現行版は第 2 世代（Studio）です。クラスベースのモジュール構造に全面書き換えし、
3 ペインエディタ（パーツ一覧／キャンバス／インスペクター）として再設計しています。

## 特長

- **直感的な編集** — 左のパーツをクリック、またはキャンバスへドラッグ＆ドロップして配置。挿入位置は X / Y 両方向で判定
- **自由なレイアウト** — 各パーツに幅（1/2・1/3・1/4・2/3・3/4）を指定して横並び配置
- **レイアウトパターン** — 「ランキング」「バナー特集」「商品紹介（左右）」をワンクリック挿入（Undo 1 回で取り消し可能）
- **豊富なパーツ** — ヒーロー／見出し（5 種のデザイン）／テキスト／画像／ボタン／画像＋テキスト／スペーサー／商品カード／バナーグリッド／レビュー／バッジ／価格／注意事項／配送情報／生産者紹介
- **バナーグリッド** — 画像 URL・遷移先リンク・キャプション・価格を設定し、均等グリッド／ランキング型（1 件目を大きく）で表示。順位バッジ・画像比率固定（1:1 / 4:3 / 16:9）にも対応
- **コード出力 2 方式** — 「べた書き（インライン）」と「style タグ集約（class 再利用で容量削減）」を切り替え可能
- **CMS 耐性** — 出力コードは改行なしの 1 行（CMS の改行→`<br>`自動変換対策）。`bgk-` 接頭辞のスコープリセット CSS を同梱し、CMS テーマの CSS 汚染・`<br>`注入からレイアウトを防御
- **レスポンシブ** — PC / タブレット / スマホのプレビューと、パーツ単位のデバイス別上書き（幅・文字サイズ・余白）。モバイルでは 2 列化・全幅化を自動適用
- **画像管理** — 画像をアップロードしてキャンバスで使用し、CMS 登録後の本番 URL を設定すると出力時に自動置換
- **その他** — Undo / Redo（スナップショット式）、自動保存（localStorage）、プロジェクトの JSON 保存／読込、単体 HTML 出力、SEO 設定（title / description / keywords / OGP / favicon）

依存ライブラリなしのバニラ JavaScript（HTML / CSS / JS）で構成しています。

## 使い方

### そのまま開く
`index.html` をブラウザで開くだけで動作します。

### ローカルサーバーで開く（推奨・Windows）

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\.claude\serve.ps1
```

起動後、ブラウザで <http://localhost:8420> を開きます。

### コードの出力
1. パーツを配置してページを作成
2. ツールバーの「コード出力」をクリック
3. 「べた書き」または「style タグ集約」を選び、「コピー」
4. CMS の HTML 編集欄に貼り付け

単体で表示できる HTML ファイルが必要な場合は、同じモーダルの「HTML ダウンロード」を使用します。

## ディレクトリ構成

```
ベタガキ君/
├─ index.html   … 画面（3 ペインシェル）
├─ style.css    … デザインシステム（ライトテーマ / トークン方式）
├─ app.js       … 基盤層（EventBus / DocumentModel / HistoryManager / AppCore）
├─ editor.js    … エディタ層（ComponentRegistry / Canvas / DragDrop / Inspector）
└─ storage.js   … 保存・出力層（AutoSave / Import / Export / 画像管理）
```

すべてのコンポーネントは `editor.js` の定義（defaults / fields / html()）で駆動され、
キャンバス描画と CMS 出力が同一の `html()` を共有します。

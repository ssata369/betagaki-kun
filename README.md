# ベタガキ君

ノーコードで LP・EC コンテンツを作成し、**CMS にそのまま貼り付けられる HTML コード**を出力するブラウザツールです。
エンジニアでなくても、パーツをクリック／ドラッグして組み立て、出力したコードを CMS の所定欄に貼るだけでページを再現できます。

## 特長

- **直感的な編集** — 左のパーツをクリック、またはキャンバスへドラッグ＆ドロップして配置
- **自由なレイアウト** — 各パーツに幅（1/2・1/3・1/4・2/3・3/4）を指定して横並び配置
- **レイアウトパターン** — 「ランキング」「バナー特集」「商品紹介（左右）」をワンクリック挿入
- **豊富なパーツ** — 見出し（5種のデザイン）／テキスト／画像／ボタン／画像＋テキスト／バナー一覧／商品カード／価格／レビュー／バッジ／注意事項／配送情報／生産者紹介／ヒーロー
- **バナー一覧** — 画像 URL・遷移先リンク・キャプション・価格を設定し、均等グリッド／ランキング型（1件目を大きく）で表示。順位バッジにも対応
- **コード出力 2 方式** — 「べた書き（インライン）」と「style タグ集約（class 再利用で容量削減）」を切り替え可能
- **レスポンシブ** — PC / タブレット / モバイルのプレビューとデバイス別スタイル
- **その他** — Undo / Redo、自動保存（localStorage）、プロジェクトの保存／読込、SEO 設定

依存ライブラリなしのバニラ JavaScript（HTML / CSS / JS）で構成しています。

## 使い方

### そのまま開く
`index.html` をブラウザで開くだけで動作します。

### ローカルサーバーで開く（推奨・Windows）
一部機能を安定して動かすため、簡易サーバー経由での起動を推奨します。

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\.claude\serve.ps1
```

起動後、ブラウザで <http://localhost:8420> を開きます。

### コードの出力
1. パーツを配置してページを作成
2. ツールバーの「コード出力」をクリック
3. 「べた書き」または「style タグ集約」を選び、「コピー」
4. CMS の HTML 編集欄に貼り付け

単体で表示できる HTML ファイルが必要な場合は「HTML 出力」を使用します。

## ディレクトリ構成

```
ベタガキ君/
├─ index.html          … 画面
├─ app.js              … 初期化・全体制御
├─ style.css           … スタイル
├─ assets/             … 画像アセット
└─ js/
   ├─ state.js         … 状態管理
   ├─ render.js        … キャンバス描画
   ├─ sidebar.js       … パーツ一覧
   ├─ property.js      … プロパティ編集
   ├─ drag.js          … ドラッグ＆ドロップ・パーツ追加
   ├─ patterns.js      … レイアウトパターン
   ├─ section.js       … セクション管理
   ├─ exporter.js      … コード / HTML 出力
   ├─ history.js       … Undo / Redo
   ├─ autosave.js      … 自動保存
   ├─ project.js       … プロジェクト保存 / 読込
   ├─ responsiveEditor.js / device.js / preview.js … レスポンシブ
   ├─ imageManager.js / seo.js / template.js / utils.js
```
